# Leap Motion Tracker with React

## Getting started

You need [Leap Motion](https://www.leapmotion.com/).

```bash
git clone git@github.com:kashira2339/leap-motion-with-react.git
cd leap-motion-with-react
npm start
# Go to localhost:3000
```
